This should be the Login UI

You ended up here because you did not set the following configuration value:

"path/to/kratos/config.yml
selfservice:  flows:    login:      ui_url: http://my-app.com/login

You can set this configuration value using environment variable SELFSERVICE_FLOWS_LOGIN_UI_URL as well!

If you don't know what that means, head over to Self-Service User Login!
