Scaling ORY Hydra

ORY Hydra is running in high-scale production environments that do millions of requests per day. To scale ORY Hydra, simply spin up another VM / Docker Container / Pod of ORY Hydra with the same configuration. There is no need for complex key/value stores or message queues to serve high traffic environments!

If you use multiple SQL instances make use of HAProxy or similar technology to do SQL Load Balancing.
