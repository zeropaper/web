Community

Ory is possible because of a unique, experienced, and dedicated community. Sponsors, contributors, and adopters make this ecosystem possible and shape it in significant ways.

Important community resources include GitHub, Slack (chat) and GitHub Discussions for Ory Kratos, Ory Hydra, Ory Keto, Ory Oathkeeper and all other Ory projects, you can also visit our archived forum.
